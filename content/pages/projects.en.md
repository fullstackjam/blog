List of projects that I am proud of.

- [OpenBoot](https://github.com/openbootdotdev/openboot) — One-command Mac dev environment setup, a Go-powered TUI for packages, dotfiles, macOS preferences, and team sharing
- [lark-coding-agent-bridge](https://github.com/fullstackjam/lark-coding-agent-bridge) — Bridge Feishu/Lark messenger to local coding agents (Claude Code / Codex / opencode) with streaming cards, session management, and multi-workspace support
- [k8s-gitops](https://github.com/fullstackjam/k8s-gitops) — Production-grade Kubernetes homelab infrastructure with PXE bare-metal provisioning, ArgoCD GitOps, Harbor registry, Grafana/Prometheus monitoring, and Ceph distributed storage
- [dotfiles](https://github.com/fullstackjam/dotfiles) — Configuration files managed with GNU Stow, paired with [OpenBoot](https://openboot.dev) for one-command dev environment setup
