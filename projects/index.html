我引以为豪的项目列表。

  • OpenBoot — 一条命令搞定 Mac 开发环境，Go 编写的 TUI 工具，支持软件安装、dotfiles、macOS 配置、团队共享
  • lark-coding-agent-bridge — 飞书连接本地 Coding Agent（Claude Code / Codex / opencode）的桥接服务，支持流式卡片、会话管理、多工作区
  • k8s-gitops — 生产级 Kubernetes homelab 基础设施，PXE 裸金属部署、ArgoCD GitOps、Harbor 镜像仓库、Grafana/Prometheus 监控、Ceph 分布式存储
  • dotfiles — GNU Stow 管理的配置文件，搭配 OpenBoot 一键初始化开发环境
